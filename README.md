# ma-petite-draft-app
 Soccer Draft Application for the Fantasy Game "Mon Petit Gazon"

#To run test 
cd ma-petite-draft-client
npm run test:unit

#To run the server: 
cd ma-petite-draft-server
node app.js

#To run the front end:
cd ma-petite-draft-client
npm run serve

#Then go to http://localhost:3001/home
